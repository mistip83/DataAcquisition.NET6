Welcome to Data Acquisition App

This is a .NET5 + Angular 12 multi-layered web application to practicing on implementing C# design patterns and fundamental Angular concepts such as components, modules, routing and data binding.

Web UI project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 12.2.2.

What does this application do?

This application allows users to collect data from different type of data loggers and display experiment results on UI.

Scenario

Each data logger device belongs to a workstation, workstations belong to a facility, facilities belong to a company.

(Image source: www.keysight.com)

Users can add new facilities, workstations and devices to create experiments regarding their companies' organization structure.

Installation

1. Backend

1. Choose DataAcquision.API as a startup project.

2. In Package Manager Console, run `Update-Database` command. This will create local database and will apply seed data.

2. Frontend

1. Install Nodejs LTS `14.16.1`

2. On VS Code terminal or Powershell, go to acquisition-web-app folder and run `npm install` command.

Start the application

1. Start DataAcquision.API, you will see the swagger document.

2. In Angular project, run `ng serve` command and navigate to `http://localhost:4200/`.
